# color_mixer

A device to mix colors. 

The potentiometer is used to set the intensity of a colour and the button is used to set the value and switch to the next colour. 

## Parts Used

- 1 Arduino UNO
- 1 Breadboard
- 1 usb cable
- 1 RGB LED
- 1 Potentiometer
- 1 push button
- 9 m-m wires
- 3 220 ohm resistors
