# LED-binary-counter

Uses LEDs to display binary numbers from 0 to 15. Can be extended to any binary number by connecting more LEDs and adding a little bit of code. 

## Parts used
- 1 Arduino UNO
- 1 breadboard
- 1 usb cable
- 4 Blue LED
- 4 220 ohm resistors
- 9 M-M cables
